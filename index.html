Symboles BPMN de Base

Catégorie Description
Contexte Piscines et couloirs pour représenter les acteurs et les responsabilités
Événements Représentent le début, la fin ou un point intermédiaire d'un processus
Activités Les tâches ou sous-processus à réaliser
Passerelles Points de décision et de convergence dans le processus
Flux Connectent les éléments et montrent l'ordre d'exécution
Données Objets de données et magasins de données utilisés dans le processus

Événements BPMN

Type Description
Événement de début générique Marque le début d'un processus
Événement de fin générique Marque la fin d'un processus
Événement de début message Le processus démarre par une demande d'un acteur externe
Événement de fin message Le processus se termine par l'envoi d'un message à un acteur externe
Événement intermédiaire d'émission message Envoi d'un message à un acteur externe
Événement intermédiaire de réception message Attente d'un message d'un acteur externe pour continuer
Événement de début minuterie Le processus démarre à un moment spécifique
Événement intermédiaire minuterie Le processus est en pause jusqu'à un moment spécifique

Activités BPMN

Type Description
Activité générique Une tâche générique sans type spécifique
Activité manuelle Effectuée par une personne sans l'usage d'un système informatique
Activité utilisateur Effectuée par une personne à l'aide d'un système informatique
Activité service Effectuée par un système informatique sans intervention humaine

Passerelles BPMN

Type Description
Passerelle exclusive Un point de décision où une seule sortie est possible (OU exclusif)
Passerelle exclusive de bifurcation Dirige le flux vers une seule voie en fonction d'une condition
Passerelle exclusive de convergence Rassemble plusieurs flux alternatifs en un seul

Flux BPMN

Type Description
Flux de séquence Montre l'ordre des activités à réaliser
Flux de message Montre les échanges d'information entre différentes piscines (acteurs externes)
Association Relie des éléments (activités, événements) à des objets de données ou magasins de données

Éléments de Données BPMN

Type Description
Objet de données Information spécifique traitée par une activité (document, formulaire, etc.)
Magasin de données Endroit où le processus peut lire et écrire des données persistantes (base de données, classeur, etc.)
Association - Lecture L'activité consulte l'information de l'objet ou du magasin de données
Association - Création L'activité crée ou écrit dans l'objet ou le magasin de données
Association - Mise à jour L'activité modifie l'information existante dans l'objet ou le magasin de données
